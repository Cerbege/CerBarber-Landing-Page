# cerBARBER - Modern Barber Landing Page

A premium, modern, and pixel-perfect landing page designed for **cerBARBER**, reflecting a luxury and sleek barbershop atmosphere. This project is built completely from scratch using clean, standard layout techniques without any heavy frameworks.



## ✨ Features
* **Pixel-Perfect & Clean UI:** A luxury dark-themed design crafted with typography and gold-accented color palette.
* **100% Fully Responsive Layout:** Seamlessly scales across all screen breakpoints, from large desktop monitors down to mobile screens (480px) using advanced CSS Grid and Flexbox.
* **No-Distortion Image Grids:** Features unified image aspect ratios (`aspect-ratio: 4 / 3`) with smart cropping (`object-fit: cover`) ensuring consistent aesthetics.
* **Smooth Hover Interactions:** Elegant micro-interactions and scale effects on feature cards and navigation links for enhanced user experience.
* **Modern CSS Techniques:** Utilization of responsive typography via `clamp()` and zero overflow issues.

## 🛠️ Tech Stack & Standards
* **HTML5:** Semantic elements used for better SEO, accessibility, and clean structure (`<header>`, `<nav>`, `<section>`, `<footer>`).
* **CSS3:** Custom styles leveraging CSS Grid, Flexbox, layout formulas, and Media Queries.
* **Google Fonts:** Integrated with elegant typography (*Montserrat* & *Playfair Display*).
* **Favicon Integration:** Custom branding icon embedded into the browser tab.
